Instructions
**Note: Send token with header. Authorization type must be Bearer
- To get all product, send a GET request here api/products
- For product details , send a GET request here api/products/{product_id}
- For product search , send a POST request here api/products/search
- For register a customer send a POST request here api/user/registration
- For customer login send a POST request here api/user/login
- For buying product, checkout order send POST request here api/orders/checkout
- For showing logged in customer order list send POST request here api/orders

Get All Product
Url: api/products
Method: GET
Returned Response Example
 
            {
                "products":
                            [
                               {
                                    'product_id': 5,
                                    'image': image path,
                                    'product_name': Game box 09A,
                                    'price': 400,
                               },
                            ]
            }

Product Details
Url: api/products/{product_id}
Method: GET
Returned Response Example
 
            {
                "product":
                           {
                                'product_id': 5,
                                'image': image path,
                                'product_name': Game box 09A,
                                'price': 400,
                           },
            }

Product Search

Request Parameters
PARAMETER NAME DESCRIPTION
search_key Send request with parameter(search_key), this key will be search product list by product name
Url: api/products/search
Method: POST
Returned Response Example
 
            {
                "products":
                            [
                               {
                                    'product_id': 5,
                                    'image': image path,
                                    'product_name': Game box 09A,
                                    'price': 400,
                               },
                            ]
            }
